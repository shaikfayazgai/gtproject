**DS-17: Avatar** (`avatar/`)
```typescript
'use client'
import { Avatar as RadixAvatar } from 'radix-ui'
import { cva, type VariantProps } from 'class-variance-authority'
import { cn } from '../../lib/utils'
```

Size variants via CVA:
```typescript
const avatarVariants = cva('relative flex shrink-0 overflow-hidden rounded-full', {
  variants: {
    size: {
      sm: 'h-8 w-8 text-xs',
      md: 'h-10 w-10 text-sm',
      lg: 'h-12 w-12 text-base',
      xl: 'h-16 w-16 text-lg',
    },
  },
  defaultVariants: { size: 'md' },
})
```

Three modes controlled by props:
1. **Image mode** (default): `src` prop provided -- renders RadixAvatar.Image with RadixAvatar.Fallback showing initials
2. **Initials mode**: No `src` -- renders RadixAvatar.Fallback with initials derived from `name` prop (first letter of first two words). Fallback bg: `bg-brand-primary text-white font-body font-medium`
3. **Anonymous mode**: `anonymous` boolean prop -- renders a deterministic SVG shape instead of image or initials. The shape is generated from a `seed` prop (string, typically user ID). Implementation:
   - Hash the seed string to a number (simple char code sum modulo)
   - Select from 6 predefined SVG shapes: circle-pattern, diamond, hexagon, triangle-pair, cross, waves
   - Select fill color from palette: brand-primary, brand-sand, brand-forest, brand-teal, brand-gold, status-success
   - Render the selected shape as an inline SVG inside the avatar circle
   - The SVG shapes should be simple geometric patterns (3-4 SVG elements each)

Exports: `Avatar`, `AvatarImage`, `AvatarFallback`, plus the main `Avatar` wrapper component that handles all 3 modes.

Story: Grid showing all sizes, image mode, initials mode (various names), anonymous mode (5 different seeds showing different shapes/colors).
