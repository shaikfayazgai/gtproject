Follow the exact code pattern from 05-RESEARCH.md "Recharts Gantt Timeline" code example. Key implementation details:

```typescript
'use client'
import { useMemo } from 'react'
import {
  BarChart, Bar, XAxis, YAxis, Tooltip, ResponsiveContainer,
  ReferenceLine, Cell,
} from 'recharts'
import { format } from 'date-fns'
import type { ProjectMilestone } from '@glimmora/types'
```

**Data transformation:**
```typescript
const data = useMemo(() =>
  milestones.map((m) => {
    const mStart = new Date(m.targetDate).getTime()
    const mEnd = m.completedDate
      ? new Date(m.completedDate).getTime()
      : mStart + 7 * 24 * 3600 * 1000 // Default 1-week duration for pending
    return {
      name: m.name,
      offset: mStart - startTs,     // invisible offset bar
      duration: mEnd - mStart,       // visible duration bar
      status: m.status,
    }
  }),
  [milestones, startTs]
)
```

**Health color mapping:**
```typescript
const healthColors: Record<string, string> = {
  completed: 'var(--color-status-success)',
  'in-progress': 'var(--color-brand-primary)',
  pending: 'var(--color-border)',
  overdue: 'var(--color-status-urgent)',
}
```

**Chart structure:**
- ResponsiveContainer with dynamic height based on milestone count
- BarChart with layout="vertical" and barSize={20}
- XAxis type="number" domain={[0, endTs - startTs]} with tickFormatter: `(val) => format(new Date(startTs + val), 'MMM d')`
- YAxis type="category" dataKey="name" width={160}
- Tooltip with custom formatter that hides offset bar value and shows duration in days
- Bar dataKey="offset" stackId="gantt" fill="transparent" (invisible spacer)
- Bar dataKey="duration" stackId="gantt" radius={[4,4,4,4]} with Cell for per-bar coloring
- ReferenceLine at today's offset position with dashed stroke and "Today" label (only if today is within project date range)

**Tooltip customization:**
- Use contentStyle with CSS variables: bg-bg-card, border-border, radius-inner
- formatter: if name === 'offset' return null; otherwise show duration in days
- labelStyle with text-text-heading
